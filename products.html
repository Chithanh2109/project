Chú ý: Đây là file HTML tĩnh không phải là giao diện web đầy đủ của ứng dụng.

Bạn đang mở file HTML tĩnh trực tiếp từ thư mục, vì vậy bạn sẽ không thấy giao diện đầy đủ và chức năng của trang web.

Để xem trang web đúng cách:

  1. Mở cmd hoặc PowerShell tại thư mục gốc của dự án
  2. Chạy lệnh: mvn spring-boot:run
  3. Đợi ứng dụng khởi động (khoảng 30 giây đến 1 phút)
  4. Mở trình duyệt và truy cập: http://localhost:8080/products

Khi đó bạn sẽ thấy giao diện đầy đủ với dữ liệu sản phẩm và có thể tương tác với trang web.

Hoặc nếu bạn chỉ muốn xem demo giao diện (không có dữ liệu), bạn có thể xem file HTML trong thư mục:

src/main/resources/templates/products.html
